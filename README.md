# cpp-docker-stack

This project is a setup for C++23 server applications with the following stack

```markdown
Project: C++ Server Application
├── 📦 Core Application
│   ├── 🌐 Language: C++23
│   └── 📚 Libraries:
│       └── Boost (Asio, Beast)
├── 🐳 Containerization (Docker)
│   ├── 🚀 App Service (C++ Server)
│   └── 📀 Database Service
│       └── PostgreSQL
└── 🛠️ Build System
└── CMake
```
